主な機能

📱

NFCタグ読み取り

トミカにNFCタグを貼り付けて、スマホでタッチするだけで情報を読み取り。簡単に持ち出し・帰宅管理ができます。

🔊

音声フィードバック

NFCタグを読み取ると「ピッ！」という音が鳴り、子どもが楽しく片付けられます。PayPayのような親しみやすい音声です。

📊

管理ダッシュボード

トミカの状態を一目で確認。帰宅中・外出中・行方不明（48時間以上）の状態を色分けで表示します。

🔍

検索・一覧機能

登録したトミカを名前で検索したり、一覧で確認可能。購入日やメモも記録できます。

⚠️

行方不明検知

48時間以上帰宅していないトミカを自動で「家出中」として検知。大切なトミカを見つけやすくします。

📝

簡単登録・編集

新しいトミカの登録や既存の情報編集が簡単。子どもでも操作しやすいシンプルなインターフェース。

使い方

1

NFCタグを貼り付け

トミカにNFCタグを貼り付けます。小さなシール状のタグなので、トミカの裏側に貼るだけでOK。

2

アプリで新規登録

アプリを開いて「新規登録」をタップ。トミカの名前や購入日を入力してNFCタグを読み取ります。

3

お出かけ時にタッチ

外出する時は持ち出すトミカをスマホでタッチ。「ピッ！」という音と共に外出状態に変わります。

4

帰宅時にもタッチ

帰宅したら再度タッチして帰宅状態に。忘れものがないか一覧で確認できます。

技術情報

📱 React Native

iOS・Android両対応のクロスプラットフォームアプリ。Expo Router を使用した高速開発。

🏷️ NFC対応

NTAG213 NFCタグに対応。react-native-nfc-manager による安定した読み取り性能。

🗄️ Supabase

PostgreSQLベースの高性能データベース。リアルタイムデータ同期とセキュアな認証。

🎨 モダンUI

Linear Gradient と Expo Vector Icons を使用した美しいインターフェース。

今後の展望

👥

家族共有機能

Google認証による家族アカウント連携で、複数の子どもでトミカを共有管理。

📷

バーコード対応

購入時のバーコードからトミカ情報を自動取得して、登録作業をより簡単に。

🎯

ARマーカー

NFCタグの代替としてARマーカーを使用。複数のトミカを同時に認識可能。

🎵

音声強化

トミカの名前を音声で読み上げる機能で、より楽しい片付け体験を提供。
